SNIPPETS MANAGER AYUDA

COMANDO RENAME

Permite renombrar archivos y categorias

Comando y Parametros

rename
[-to]

DESCRIPCION

Este comando permite renombrar archivos, categorias y subcategorias.

OPCIONES

  • [-to]: Este parametro requiere el nuevo nombre

EJEMPLOS

Renombrar un archivo o carpeta:

  • SnippetsM> rename "default/snippet.txt" -to "snippetNuevo.txt"
  • SnippetsM> rename
Archivo a renombrar> "default/snippet.txt"
Nombre nuevo> "snippetNuevo.txt"
